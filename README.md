## 项目名称
> 同步你的目标git仓库到多个不同的仓库中例如：codeup->github\gitlab\gitee



## 运行条件
> 列出运行该项目所必须的条件和相关依赖  
* 条件一：在.netrc文件中配置你的git相关主机地址、用户名、密码
* 条件二：你的环境中需要安装jq工具、对json配置文件进行解析
* 条件三：使用前检查脚本运行权限



## 运行说明
> 说明如何运行和使用你的项目，建议给出具体的步骤说明
* 操作一：配置git-sync-config.json文件。说明：name：仓库名称（必须），type：类型（必须），url：仓库地址（必须），sync-list：需要同步的仓库列表（必须）
* 操作二：赋予脚本执行权限 chmod +x git.sh
